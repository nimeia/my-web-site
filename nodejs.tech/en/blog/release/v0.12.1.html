Node v0.12.1 (Stable)

2015-03-24

This release upgrades OpenSSL to version 1.0.1m, which includes fixes for several CVEs that were disclosed recently.

2015.03.23, Version 0.12.1 (Stable)

  • openssl: upgrade to 1.0.1m (Addressing multiple CVES)
