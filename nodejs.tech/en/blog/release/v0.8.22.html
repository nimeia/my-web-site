Node v0.8.22 (Stable)

2013-03-07

2013.03.07, Version 0.8.22 (Stable)

  • npm: Update to 1.2.14

  • cluster: propagate bind errors (Ben Noordhuis)

  • crypto: don't assert when calling Cipher#final() twice (Ben Noordhuis)

  • build, windows: disable SEH (Ben Noordhuis)
