Node v11.1.0 (Current)

By Michaël Zasso, 2018-11-02

Notable changes

  • deps
    • Updated ICU to 63.1. #23715
  • repl
    • Top-level for-await-of is now supported in the REPL. #23841
  • timers
    • Fixed an issue that could cause timers to enter an infinite loop. #23870
