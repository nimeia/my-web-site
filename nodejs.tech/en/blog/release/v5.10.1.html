Node v5.10.1 (Current)

By Myles Borins, 2016-04-06

Notable changes

  • http:

    • Enclose IPv6 Host header in square brackets. This will enable proper separation of the host address from any port reference (Mihai Potra) #5314

  • path:

    • Make win32.isAbsolute more consistent (Brian White) #6028
