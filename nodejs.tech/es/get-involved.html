Participe

Discusión de la Comunidad

  • La lista de errores es el lugar para discutir las características del núcleo de Node.js.
  • La cuenta de Twitter oficial de Node.js es nodejs.
  • El calendario de la Fundación Node.js con todas las reuniones del equipo público.
  • Node Weekly es una lista de correo que recopila los últimos eventos y noticias alrededor de la comunidad de Node.js.
  • Node Slackers es una comunidad de Slack enfocada en Node.js.

Aprendizaje

  • La Documentación oficial de la API detalla la API de Node.
  • NodeSchool.io le enseñará conceptos de Node.js de forma interactiva mediante juegos utilizando la línea de comandos.
  • La etiqueta de Node.js en Stack Overflow colecciona nueva información cada día.
  • La etiqueta DEV Community Node.js es un lugar para compartir proyectos, artículos y tutoriales de Node.js, así como para iniciar debates y pedir información sobre temas relacionados con Node.js. Desarrolladores de todos los niveles son bienvenidos a participar.
  • Nodeiflux es una comunidad amigable de desarrolladores back-end de Node.js que se apoyan mutuamente en Discord.
