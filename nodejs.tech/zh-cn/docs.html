API 函数接口文档

API 函数接口文档 提供了在 Node.js 中的函数或者对象的详细信息。这个文档告诉你什么方法该用什么参数，方法返回值是什么以及使用那个方法会有什么错误异常。它同时也告知你不同版本中的 Node.js 有哪些方法是可用的。

此文档描述了 Node.js 内建的模块，社区提供的模块并不记载在内。
