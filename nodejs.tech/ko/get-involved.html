참여하기

커뮤니티 논의

  • GitHub 이슈 목록에서 Node.js 코어 기능에 대한 논의가 이루어집니다.
  • Node.js의 공식 트위터 계정은 nodejs입니다.
  • Node.js 재단 캘린더에 모든 공개 팀 미팅 일정이 포함되어 있습니다.

알아보기

  • 공식 API 레퍼런스 문서는 Node.js API에 대해 설명합니다.
  • NodeSchool.io는 대화형 명령행 게임을 통해 Node.js 개념에 대해 알려줍니다.
  • Stack Overflow Node.js 태그에는 매일 새로운 정보가 쌓입니다.
  • The DEV Community Node.js 태그는 Node.js와 관련된 주제에 대해 논의하거나 의견을 묻기에도 좋고, Node.js 프로젝트, 글, 튜토리얼을 공유하기에도 좋은 장소입니다. 개발자라면 모두 환영합니다.
  • Nodeiflux는 Discord에서 서로를 지원하는 Node.js 백엔드 개발자 커뮤니티입니다.
