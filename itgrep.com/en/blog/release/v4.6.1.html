Node v4.6.1 (LTS)

By Rod Vagg, 2016-10-19

This is a security release. All Node.js users should consult the security release summary at https://nodejs.org/en/blog/vulnerability/october-2016-security-releases/ for details on patched vulnerabilities.

Notable Changes

  • c-ares: fix for single-byte buffer overwrite, CVE-2016-5180, more information at https://c-ares.haxx.se/adv_20160929.html (Daniel Stenberg)

Commits

  • [f3c63e7ccf] - deps: avoid single-byte buffer overwrite (Daniel Stenberg) #8849
  • [5a0daa6c2f] - win,build: try multiple timeservers when signing (Rod Vagg) #9155
