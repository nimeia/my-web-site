Participe

Discussões da Comunidade

  • A lista de problemas do GitHub é o lugar para discutir as principais funcionalidades do Node.js.
  • A conta oficial do Node.js no Twitter é nodejs.
  • O calendário da Fundação Node.js contém todas as reuniões dos times públicos

Aprenda

  • Documentação oficial da API detalha a API do Node.js.
  • NodeSchool.io ensinará os conceitos do Node.js através de jogos interativos em linha de comando.
  • Tag Node.js no Stack Overflow coleta novas informações todos os dias.
  • Tag Node.js na DEV Community é o lugar para compartilhar projetos, artigos e tutoriais, bem como iniciar discussões e solicitar feedback sobre tópicos relacionados ao Node.js. Desenvolvedores de todos os níveis são bem-vindos a participar.
  • Nodeiflux é uma comunidade de desenvolvedores backend que utilizam Node.js, auxiliando uns aos outros através do Discord.
