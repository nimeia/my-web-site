Обсуждение сообщества

  • Список проблем на GitHub это место для обсуждения возможностей ядра Node.js.
  • Официальный аккаунт Node.js в Твиттере ― nodejs.
  • Календарь Фонда Node.js со всеми общественными встречами.

Обучение

  • Официальная справочная документация по API описывает API Node.js
  • NodeSchool.io научит вас концепциям Node.js через интерактивные игры командной строки.
  • Тег Node.js на Stack Overflow собирает новую информацию каждый день.
  • Тег сообщества на DEV.js ― это место, где можно делиться проектами, статьями и учебными пособиями по Node.js, а также начинать обсуждения и запрашивать отзывы по темам, связанным с Node.js. Разработчики всех уровней квалификации могут принять участие.
  • Nodeiflux, является дружественным сообществом бекенд-разработчиков на Node.js, поддерживающих друг друга на Discord.
