API 參考文件

API 參考文件提供了 Node.js 中的函式或物件的詳細資訊。此文件詳述了一個方法可接受哪些參數、其回傳值及哪些錯誤可能與之關聯，同時也記錄了在哪些 Node.js 版本中可使用此方法。

此文件僅會介紹由 Node.js 官方內建的模組，社群提供的模組則不在此列。
