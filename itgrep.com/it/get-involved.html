Come partecipare

Comunità

  • La lista delle issues di GitHub è il luogo in cui discutere delle funzionalità del core di Node.js.
  • L'account Twitter ufficiale di Node.js é nodejs.
  • Il calendario della Node.js Foundation con tutti i meeting pubblici del team.
  • Node Slackers è la comunità Slack di Node.js.

Impara

  • Documentazione ufficiale delle API contiene i dettagli delle API di Node.js.
  • NodeSchool.io insegna i concetti di Node.js attraverso giochi interattivi da linea di comando.
  • Il tag Node.js su Stack Overflow colleziona molte informazioni ogni giorno.
  • Il tag Node.js nella comunità degli sviluppatori é il luogo dove condividere i progetti Node.js, articoli e tutorial come anche discussioni e per fare domande relative ad argomenti su Node.js. Sono benvenuti tutti gli sviluppatori di qualsiasi livello.
  • Nodeiflux é una comunità di sviluppatori backend di Node.js con supporto tramite Discord.
