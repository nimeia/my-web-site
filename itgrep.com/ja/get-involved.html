コミュニティディスカッション

  • GitHub issues リスト は、Node.js のコア機能に関する議論の場です。
  • Node.js 公式 Twitter アカウントは nodejs です。
  • Node.js Foundation calendar とすべての公開チームミーティング。

学習

  • 公式 API リファレンスドキュメント は、Node.js API の詳細を説明しています。
  • NodeSchool.io はインタラクティブなコマンドラインゲームを介して Node.js の概念を教えます。
  • Stack Overflow Node.js tag は、毎日新しい情報を収集します。
  • The DEV Community Node.js tag は、Node.js プロジェクト、記事、チュートリアルを共有したり、ディスカッションを開始したり、Node.js 関連のトピックについてのフィードバックを求めたりする場所です。スキルレベルに関係なくすべての開発者が参加できます。
  • Nodeiflux は、Discord でお互いをサポートする Node.js バックエンド開発者のフレンドリーなコミュニティです。
